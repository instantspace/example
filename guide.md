## 웹개발과정(front+back)

# 프로젝트를 위한 선택지

```
1.서버선택
2.백엔드언어선택
3.데이터베이스선택
4.아키텍트만들기
5.프론트엔드(spa,링크방식)선택
```

# 웹개발 기술사항

```
riahnrhyme
1. os선택 (윈도우서버,리눅스(취업잘됨 추천))
    -윈도
    -센토스centos(취업추천)(유료로 변경하면 기술지원받음)
    -ubuntu(개인작업용 추천)
    
2.웹서버(웹언어를 해석할수있게 도와줌/서버의 역할) 
    -apache(취업추천)
    -nginx(최근 뜨는중)
    -node(자바스크립트기반.프론트가 풀스택을 할수있게 도와줌)
        ㄴsocket io(채팅제작)
3.백엔드
    -php
    -phython(추천)
    -java / jsp(취업상추천)
    -node
    -go lang(구글제작언어)
    
4.데이터베이스
    -oracle DB (접하기 어려움)
    -mysql / mariadb (취업추천)
     (유료)   (무료)
    -mongodb(nosql) -명령어,규칙등없음

5.개발방식
    -restful api(취업추천,여러디바이스에 제한이 없음)
    -mvc(model-controler-view)
    
6.프론트엔드
    -html,css,js(필수)
7.프론트방식
    -spa(react(매니악), angular, vue(비전공자에게 쉬움))
    -페이지이동방식
8.ui / ux
    -반응형
    -하이브리드 (웹+모바일웹)
    -네이티브(웹+앱)

```
